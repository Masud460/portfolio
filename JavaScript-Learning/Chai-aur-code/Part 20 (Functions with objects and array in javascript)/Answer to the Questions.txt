1. To solve the problem, we can use the rest operator (...para) in function parameter's area like this:
    function calculateCartPrice(...prices) {
        return prices;
    }
    console.log(calculateCartPrice(2000, 6000, 3000)) // returns [ 2000, 6000, 3000 ]
